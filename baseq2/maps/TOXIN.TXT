* Construction *

Known Bugs              : None
Build Time              : 9 sleepless days and nights
Compile Time		: 3hrs 15min


Installation            : Just run the zip and extract it to your Quake2 folder.

			  If you're having problems and want to input the
			  files manualy, here is where they go:

			  The *.wav files go into a folder called "Toxin"
			  in the quake2\action\sound

			  The *.wal files go into a folder called "Toxin"
			  in the quake2\action\textures

			  The toxin.bsp file goes into the folder "maps"
			  in quake2\action\maps
			  
			  The drksky.* files go into the folder called "env"
			  in quake2\action\env
