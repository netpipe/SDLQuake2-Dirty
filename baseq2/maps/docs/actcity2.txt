Author                  : Lari Muuriaisniemi aka CryptR
Email Address           : lmuur@dlc.fi
Web Site                : http://www.dlc.fi/~lmuur

Description             : Map for Action Quake (http://action.telefragged.com).
                          This area, overcomed by CrossoR's troops, is
                          division of Mancis City connected with Mancis Freeway
                          located at the East.
                          This map includes Casino, Park Garage building site,
                          Motel, Industry Site and Streets & Avenues surrounded
                          by Apartment Houses and realistic Urban environment.
                          Suitable for 2-8 players.

Installation            : * Unzip ACTCITY2.ZIP into root of your Action Quake
                          game folder (Quake2/Action), with Subdirectories
                          enabled (!), so that all textures and sky will be unzipped
                          into correct directory (USE FOLDER NAMES checkbox
                          in WinZip or -d switch in PKUnZip).
                                                    
Additional Credits to   : Stecki for making the sky that has been used as the base for new one.
                          A-Team for making greatest Quake2 Multiplayer mod ever.
                          id software

WHAT'S NEW              : * Several small enhancements & speedups optimizations
                            to increase playability a bit.
                          * Changed rotating Casino doors to Sliding doors,
                            so players dont get anymore stuck between the doors.
                          * Added the Lobby for Motel Mancis, check out the
                            Freeway area!
                          * Rooftops are now more easier to access. Low roof
                            at the Casino Square is now accessable too.
                          * Bit more ammo added.
