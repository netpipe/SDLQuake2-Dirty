Email Address           : Stefan.Giesen@gmx.de
Homepage                : http://www.homepages.de/sgiesen@dialup.nacamar.de/
Description             : This one is a teamplay map for Terror Quake2.
			  It's a city map with a sewer. This map is build
			  on rhcity1.bsp, but with MANY changes. This Map
			  also fits for AQ2 DM and Teamplay (just don't care
			  about the "xxx has no spawn function" errors, they
			  just don't matter :).
Requirements            : Quake2, Terror Quake2 or Action Quake2

Q2 Single Player        : Not intended (only for sightseeing)
Q2 Cooperative          : No
Q2 Deathmatch           : Not intended
Q2 CTF                  : Not intended
Single Player (TQ2)	: Not intended (sightseeing?)
Deathmatch (TQ2/AQ2)    : Yes, should be fine
Teamplay (TQ2/AQ2)	: Yep, that's it :-)
Get That Drug (TQ2)	: No - this map isn't made for GTD.
