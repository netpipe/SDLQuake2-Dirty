* Installation *

This is very important:
You MUST have AQ2 version 0.95 or higher
The BSP must end up in X:\Quake2\Action\maps directory
Textures must end up in X:\Quake2\action\textures\TTI
New sounds MUST end up in X:\Quake2\action\sound\world
