Description             : Map for Action Quake (http://action.telefragged.com).
                          City-style map with Skyscrapers and unique methods of transportation and sneaking up on opponents.

Installation            : * Unzip vertigo.zip into root of your Action Quake
                          game folder (Quake2/Action), with Subdirectories
                          enabled (!), so that all textures and sky will be unzipped
                          into correct directory (USE FOLDER NAMES checkbox
                          in WinZip or -d switch in PKUnZip).
			  All *.wal files go into Action\textures\karr
			  All *.wav files go into Action\sound\karr
                          All *.tga files go into Action\env
                          The *.bsp file goes into Action\maps
                          The *.txt file needs to be printed out and displayed prominently in your home and place of work.
