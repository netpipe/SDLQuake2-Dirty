* SPECIAL INSTRUCTIONS / ADDITIONAL INFORMATION *
--------------------------------------------------------------------------------
Notice:

If you find the jumps too hard, type 
"sv_gravity 700" in console. This will simplify
the jumps dramaticly. (the gravity is 750 at default)
You can also set gravity to 800, if you'd like
to keep the standard aq2 gravity.(You will still 
be able to do all the jumps)
